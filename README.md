## Overview

Get Sayva is the archived Squarespace website for SAYVA Real Estate, a Toronto-based real estate practice led by Nalin Sharma. The site communicates the firm's client-first philosophy ("Selfless service. Honest advice.") and provides resources for buyers, sellers, and tenants. It features property listings with performance results, buying and selling guides, a comprehensive leasing 101 section, an about page, and a contact page with Calendly integration for booking discovery calls.

## Features

- **Home Page** -- Hero section with tagline and Calendly booking CTA, three differentiator feature blocks (genuine care, negotiation expertise, West End Toronto knowledge), service options (buy/sell/talk), and featured listings with sale/lease results
- **Buy & Sell Page** -- Guidance and resources for buyers and sellers in the Toronto market
- **Leasing 101** -- Comprehensive guide to leasing in Toronto covering the full leasing process
- **Listings Page** -- Featured property listings with results (e.g., "Sold for 101% of asking price in 4 days," "Leased for 100% of asking price")
- **About Page** -- Team biography and practice philosophy
- **Contact Page** -- Contact information and inquiry form
- **Calendly Integration** -- Direct booking link for 30-minute discovery calls
- **Responsive Layout** -- Clean, mobile-friendly design with Work Sans typography

## Getting Started

This is a static HTML site. No build step is required.

```bash
# Clone the repository
git clone git@github.com:izzydoesizzy/getsayva.git
cd getsayva

# Open in your browser
open index.html
```
